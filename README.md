# Jyothi Ceramics Portfolio

A personal portfolio website showcasing ceramic artwork including mugs, bowls, vases, and printmaking pieces by Jyothi Ravilla.

## Features

- **Animated Homepage**: Full-screen slideshow featuring ceramic pieces with smooth transitions
- **Gallery Pages**: Dedicated galleries for different ceramic categories (Mugs, Bowls, Vases)
- **Printmaking Section**: Display of printmaking artwork
- **Responsive Navigation**: Collapsible sidebar navigation with expandable ceramics submenu
- **About Page**: Information about the artist
- **Blog Section**: Space for sharing thoughts and updates
- **Contact Form**: Easy way for visitors to get in touch
- **Instagram Integration**: Direct link to social media presence

## Project Structure

```
ceramics/
├── index.html           # Landing page with slideshow
├── main.html           # Main gallery homepage
├── mugs.html           # Mugs gallery
├── bowls.html          # Bowls gallery
├── vases.html          # Vases gallery
├── printmaking.html    # Printmaking gallery
├── about.html          # About page
├── blogs.html          # Blog section
├── contact.html        # Contact form
├── style.css           # Global styles
├── images/             # Image assets organized by category
└── README.md           # Project documentation
```

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Navigation Features

- Fixed sidebar navigation on all pages
- Expandable "Ceramics" submenu showing Mugs, Bowls, and Vases
- Active page highlighting
- Smooth navigation between sections

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required

## Image Organization

Images are organized by category in the `images/` directory:
- `bowls/` - Bowl and plate images
- `mugs/` - Mug images
- `vases/` - Vase images
- `printmaking/` - Printmaking artwork
- `other/` - Miscellaneous pieces
